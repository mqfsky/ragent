## 参考答案提纲

### 1. RAG 和直接调用大模型的区别

- 直接调用模型主要依赖模型已有知识
- RAG 会先从外部知识库检索相关内容，再辅助生成答案

### 2. `RetrievalEngine` 更像执行者还是调度者

- 更像调度者
- 它负责组织检索、意图、工具调用和结果整合

### 3. 如果没有 `RetrievalEngine`，聊天系统会缺什么

- 缺少统一的检索协调能力
- 缺少把知识库结果和工具结果整合成上下文的能力
- 系统会更像一个普通模型对话接口

### 4. 为什么要先检索再生成

- 因为企业私有知识通常不在模型训练数据里
- 先检索可以提高回答准确性，减少幻觉
